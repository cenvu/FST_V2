# No Scope Creep

FST MVP must stay focused.

## Current Scope

- Single source
- Single destination
- Single job
- Copy
- Verify
- SAFE TO EJECT evidence
- Detailed TXT Report V1

## Deferred

- Multi-destination
- Parallel jobs
- Database
- Cloud sync
- PDF report
- Dashboard
- Report viewer
- Equipment tracking
- Team collaboration

## Agent Rule

If an agent proposes a deferred feature, it must clearly mark it as deferred and not implement it.
